# 📸 Photo Organizer

**A simple, private, and 100% offline AI tool to organize your photos and videos.**

Photo Organizer uses smart AI to sort your messy photo library by **faces, dates, and locations** without ever uploading your data to the cloud. It runs entirely on your computer, keeping your memories private and safe.

---

## ✨ Key Features
*   **👤 Face Recognition**: Automatically groups photos of the same person.
*   **🌍 Smart Locations**: Sorts your photos by city and country using GPS data.
*   **🗓 Date Sorting**: Organizes your library into neat folders by Year, Month, and Day.
*   **🔁 Duplicate Finder**: Identifies and separates duplicate photos to save space.
*   **🛡 100% Private**: No internet required. No cloud. No tracking.
*   **↩️ Safety First**: It never changes your original photos—it only makes organized copies.

---

## 🚀 Quick Start (No Setup Required)

If you just want to use the app, follow these steps:

1.  **Download**: Get the latest **[Photo Organizer Portable ZIP](https://github.com/DharuneshBoopathy/PhotoOrganizer/releases)**.
2.  **Extract**: Right-click the ZIP file and select "Extract All".
3.  **Run**: Open the folder and double-click **`PhotoOrganizer.exe`**.

*Note: On the first run, the app will briefly set up the AI "brain" on your machine. This might take a moment but only happens once!*

---

## 🛠 How to Use

1.  **Pick Source**: Choose the folder where your messy photos are.
2.  **Pick Destination**: Choose a new folder where you want the organized copies to go.
3.  **Click Start**: Sit back and let the AI do the work!
4.  **View Report**: When finished, open the `report.html` in your output folder to see a beautiful dashboard of your organized library.

---

## 📁 Project Structure

For developers or curious users, the project is organized simply:
*   **`PhotoOrganizer.exe`**: The main application.
*   **`app/`**: Contains all the "brains," source code, and tools.
*   **`build.bat`**: A script to rebuild the application from source.
*   **`setup.bat`**: Sets up the development environment.

---

## 💻 Building from Source

If you are a developer and want to build the app yourself:

1.  Ensure you have **Python 3.13** installed.
2.  Clone this repository.
3.  Run `setup.bat` to install dependencies.
4.  Run `build.bat` to create your own executable.
